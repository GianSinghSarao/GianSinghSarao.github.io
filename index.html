1. The String Encrypter
My Goal Was To Understand How To Package A Web Page In To A Single File That The User Can Download And Use. e.g. A Downloadable HTML Ebook. I Came Across HTA Files And Found A Tutorial On How To Make A String Encrypter. I Followed To The End And Customized It. The Styling Was Light And it Didn't Look Modern Or Contemporary Like It Should Have. Now This Was A Fairly Rubbish Application That Relied On External Internet Explorer And VBScript. So I Tried To Improve It And Failed. I Eventually Gave In And Started Looking For A New Solution. At This Point I Reverted This And Called It V1. Now I Wanted Something Quick That Could Run In Any Browser Withut Dependencies Like VBScript Which Only IE Supports.
